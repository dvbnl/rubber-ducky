# Rubber Ducky Payload Repository

This repository contains a collection of payloads for the Rubber Ducky by created by DVBNL. Also the compiled *.bin files can be found in the specific payload folders.

## Repository Structure

This repository is organized by the type of payload. Each payload has its own directory which contains the payload script and a README file explaining what the payload does and how to use it.

## How to Use

To use a payload, follow these steps:

1. Navigate to the directory of the payload you want to use.
2. Read the README file for instructions on how to use the payload.
3. Load the payload onto your Rubber Ducky according to the instructions.
